# Notion2Github
Notion2Github는 Notion과 Github 사이의 동기화를 지원하는 앱 입니다.

# Usage
- start : uvicorn app:app --reload
- swagger ui : http://127.0.0.1:8000/docs
- redoc : http://127.0.0.1:8000/redoc

# Contents
1. page auto create
2. svg export
3. md auto commit
